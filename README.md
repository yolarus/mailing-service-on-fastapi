# Сервис рассылки на FastAPI

## Описание
API представляет собой микросервис, отвечающий за рассылку сообщений пользователям по почте и в телеграм-боте с одним эндпоинтом. В приложение встроен следующий функционал:
1. Полученные сообщения и пользователи сохраняются в связанные между собой таблицы в БД.
2. Получатели автоматически проходят валидацию, а также определяется передан email или chat_id.
3. Выполняется отправка сообщений на почту и в телеграм-боте с помощью Celery с возможными вариантами задержки отправки.
4. Была предпринята попытка завернуть микросервис в контейнеры с оркестрацией, пока данная часть не работает, в будущем, вероятно, будет доработано

## Установка проекта
1. Клонирование проекта из [GitHub](https://github.com/yolarus/mailing-service-on-fastapi) по HTTPS-токену или SSH-ключу.
2. Создание и заполнение файлов .env и db.env своими данными.
3. Установить на компьютер брокер Redis

## Установка зависимостей
1. Перейти в настройки Pycharm -> Setting -> Project -> Python Interpreter.
2. Добавить локальный интерпретатор с менеджером пакетов Poetry.
3. Выполнить команду в терминале PyCharm 'poetry install --no-root'.

## Запуск
1. Прописать в командной строке 'python manage.py redis-server'. 
2. Прописать в командной строке 'celery -A celery_queue.celery.celery_app worker --loglevel=INFO'. 
3. Прописать в командной строке ' uvicorn mailings.main:app --reload'.
